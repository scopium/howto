Install Open JDK

There are a few ways of installing openjdk

1

sudo apt-get update
sudo apt-get install openjdk-7-jdk
sudo apt-get install openjdk-7-jre

2

Follow instructions from the links below http://www.devsniper.com/ubuntu-12-04-install-sun-jdk-6-7/

3

sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer

4

update your JAVA_HOME in /etc/environment
source /etc/environment  to restart
sudo update-alternatives --get-selections | grep ^java
java -version
